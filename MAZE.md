##### I need a browser based application that allows the user to draw a maze, and then the computer solves the maze. 



1. The display for the maze should start out being entirely filled in. 
2. The user should be able to define the maze by opening pathways, or in other words, drilling through the closed playing field.
3. The user should be able to specify the beginning and end of the maze. 
4. It should also be possible to fill in blocks.
5. There should be a reset button to fill in all squares of the maze. 
6. There should be a field to select the search algorithm: BFS, DFS, and A\*.
7. When told to solve the maze, the computer should slowly animate the process of solving the maze.
8. Once the end point is discovered a picture of the final solution should be shown.
9. There should be a slider "control" to specify the speed of the animation.
10. On iPhones, and iPads make sure the display fits and is anchored to the display.
11. I need it to fit completely on a laptop screen without scrolling.
12. The entire program must fit in a single HTML file.  In other words all HTML, JavaScript, CSS, and any graphics all in one HTML file.
